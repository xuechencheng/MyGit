17.7 装箱转换
包括值类型在内的所有C#类型都派生自object类型。
值类型是高效轻量的类型，因为默认情况下在堆上不包括它们的对象组件。
装箱是一种隐式转换，它接受值类型的值，根据这个值在堆上创建一个完整的引用类型对象并返回对象引用。
需要装箱的一个常见场景是将一个值类型当做参数传递给一个方法，而参数类型是对象的数据类型。
int i = 12; object oi = null; oi = i;
在装箱之后，值有两份副本--原始值类型和引用类型副本。
