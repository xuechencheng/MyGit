find命令
find [PATH] [option] [action]

将过去系统上面24小时内有改动的文件列出
find / -mtime 0

寻找/etc下面的文件，如果文件日期比/etc/passwd新就列出
find /etc -newer /etc/passwd

查找/home下面属于vbird的文件
find /home -user vbird

查找系统中不属于任何人的文件
find / -nouser

查找文件名为passwd的文件
find / -name passwd

查找/var目录下文件类型为Socket的文件名有哪些
find /var -type s

找出系统中大于1MB的文件
find / -size +1000k

找出/etc下面文件名含httpd的文件
find /etc -name "*httpd*"
