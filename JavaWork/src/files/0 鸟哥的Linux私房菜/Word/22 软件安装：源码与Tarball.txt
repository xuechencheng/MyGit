linux上的软件几乎都是经过什么授权，所以几乎都提供源代码？
GPL
查看文件是否是二进制的命令
file
使用什么命令可以简化编译过程？
make
make会在当前的目录下搜索什么文件？
makefile
检测程序并自动创建makefile文件的命令是什么？
config
将软件的所有源码文件先以tar打包，然后在压缩的文件是什么？
tarball
安装和更新软件的两种方法
1，源码编译 2，二进制安装
tarball解压文件夹里面一般有什么文件提示安装
INSTALL或README
大部分tarball软件安装的命令执行方式是什么？
1,./configure 2,makeclean 3,make 4 make install
一般软件的安装目录是什么
/usr/local
一般tarball文件压缩目录是什么呢
/usr/local/src
man命令搜索的目录是什么
/usr/local/man
示例：下载ntp软件，编译并安装
如果软件有新的更新补丁了，使用过什么命令？
patch
函数库依照什么标准分为动态和静态函数库？
是否被编译到程序内部
静态函数库有哪些特点？
扩展名为.a	可以独立执行  升级难
使用静态函数库如果升级了，使用了此函数库的程序需要怎么办？
重新编译
动态函数库有哪些特点？
扩展名为.o 没有被整合到执行文件中  不能独立执行  简单：执行文件不需要重新编译
linux比较倾向于使用哪种？
动态函数库
绝大多数函数库的路径是哪里？
/usr/lib /lib
如何将动态函数库加载高速缓存当中？
1，需要改/etc/ld.so.conf文件 2，使用ldconfig命令 3，将数据记录一份在/etc/ld.so.cache中
查看passwd使用量那些函数库的命令是什么？
ldd /usr/bin/passwd
为了保证下载的文件不会被人篡改，可以验证什么？
MD5
查看文件的md5命令
md5sum
